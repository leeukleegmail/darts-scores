                User Manual

                Use this guide for setup, scoring, and admin tools while you play.

                Quick Start

                1. Use Manage Players in Select Players to manage the player list.
                2. Select who is playing and drag them into order.
                3. Choose Singles or Teams.
                4. Pick a game from Select Game and follow any start popup for that mode.
                5. Enter scores in Live Game and review Recent Games.

                Tip: each signed-in user can run one active game at a time, but different users can play concurrently in separate sessions.

                X01

                • Choose 1001, 501, 301, or 101 in the start popup.
                • Enter one total score for each turn using the keypad.
                • To win, finish on exactly zero. The player decides that their entry is valid before submitting the turn.
                • Overshooting or leaving 1 is a bust and the turn scores nothing.

                55 by 5

                • Enter one total score for each turn.
                • Only totals divisible by 5 count.
                • Counted score is converted to fives (`total / 5`).
                • Hit exactly 55 fives to win; going over is a bust.
                • Use Undo to remove the latest turn if needed.

                English Cricket

                • The Starting Roles popup lets Team A choose whether to Bat or Bowl.
                • The live view splits into a bowling panel and a batting panel.
                • For bowling, click the bull markers to record wicket marks, then submit the turn.
                • For batting, enter the total score; runs are counted from strong scoring turns.
                • In the chase, the panel shows Target and Remaining Runs.

                Noughts and Crosses

                • Each match generates a fresh board of dart targets using Big, Small, Double, and Treble segments from 1 to 20.
                • The center square is always Bullseye.
                • Tap an open square and choose whether to claim it as X or O.
                • The X/O side boxes show who is assigned each side; turn order is managed by players at the board.
                • When a game ends, use Rematch in the winner popup to restart with swapped starting side/order.

                Accounts & History

                • Admins can create users and reset passwords from User Administration.
                • The Clear History control is visible only to admins.
                • Each signed-in user gets their own active game and matching Recent Games view.
                • Non-admin players can still score games normally after signing in.
